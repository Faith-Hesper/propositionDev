### 安装
```bash
# 安装依赖
npm i
# or
npm install
```
### 运行
#### 开发模式
```bash
npm run dev
```
#### 生产模式
```bash
npm run build
```
### 功能
#### 搜索
1. 通过搜索框，输入店名搜索目标商店，点击可查看门店相关信息
#### 查询
1. 通过绘制矩形，查询范围内门店，点击可查看门店相关信息
2. 通过绘制多边形进行查询
3. 通过添加标记点，设置查询范围或查询商店名字或门店数量，查询符合条件的商店
4. 列表展示查询的结果
#### 物流配送
1. 添加配送点，设置相关参数，查询最近的商店，并在地图上展示路线
2. 可点击地图上的商店图标，查询配送路线，或服务范围
3. 在点击查询前可拖拽标记点
4. 路线生成后，可在底部查看具体路线指引，鼠标移入可在地图上展示指引信息，或者单击图层
#### 热力图
单击热力图展示热力图
#### 服务范围
单击服务范围展示所有商店服务范围
#### 拖拽和隐藏
单击隐藏可隐藏对话框，也可进行拖拽
